12月初的时候去了一趟西北省份旅游，主要是西安和山西临汾（虽然山西更常被归类到华北）。之前买的海鸥4B双反相机一直无处使用，只在西湖周围和住处附近随便拍了几张黑白胶卷的照片，这次去旅游也刚好第一次用彩色胶卷拍照。

至于出行方式，因为我的充电宝年代比较久远，没有3C认证，又不想让胶卷通过X光机，所以依旧是选择了高铁出行。火车全程是八个小时，一路跨过长江黄河，经过中原，过函谷关进入关中盆地，可以说是一条经典的“入关”路线。最后在天黑的时候抵达西安北站。

一开始我以为西安会比较冷，但是太久没有来北方，已经忘了这里的干冷并不是很可怕了，羽绒服几乎完全用不上。我定的民宿是在“南稍门”这个地方，离附近的一些景点都算比较近。在地铁站跟硅基猫碰头之后吃了附近的冰煮羊火锅。往程的奔波一天就到此结束。

= 第1天

这一天本来是想取西安博物院和陕西历史博物馆的，但是因为预约爆满所以未能成行，于是在吃完午饭之后选择直奔西安城墙。

西安城墙的贵得有点离谱，花掉了五十多块钱。但是上了城墙之后却发现实际上并没有什么内容，主要就是走一走，感受一下历史氛围。

#image("001.jpg", width: 65%)

城墙上有很多汉服拍照的游客，非常热闹。穿汉服拍照的游客里面有男有女，但是男生的汉服往往有点像是从仙侠电视剧里面走出来的，跟城墙古朴的造型很不协调，有点出戏。

#image("002.jpg", width: 65%)

西安现在的城墙是明代城墙，而明代城墙又基本上是在唐朝长安城的皇城城墙基础上修建的，所以有一个地方有一个唐朝城墙的遗址博物馆。不过岁月沧桑，只留下了一点残垣而已。

#image("003.jpg", width: 65%)

从城墙下来之后坐两站地铁就到了鼓楼钟楼，这里应该算是西安的中心了。钟楼还挺漂亮的，但是我在操作相机的时候出了一点失误，双重曝光了，只留下了一点钟楼的残影。

#image("004.jpg", width: 65%)

晚上吃完饭过后，随便在钟楼附近逛了逛街，硅基猫看到了一个恐怖元素的密室逃脱体验馆，提议去玩一下。然后我果然被吓得出来之后很久还在发抖。因为不喜欢这类恐怖体验，所以不太想回忆了。不过说实话他们的体验效果做得确实很厉害，可以理解为什么很多人就是很喜欢玩这个。

第一天就是这样。

= 第2天

这一天的主要安排是参观秦始皇陵兵马俑博物馆。吸取了昨天预约不到博物馆门票的教训，这次我早早就做好了预约，不过兵马俑的门票似乎并不是很紧张的样子，实际上并不需要预约。

兵马俑的付费导游讲解市场竞争非常激烈，从地铁下来上公交车开始，就有各类组织开始推销导游讲解了。不过最权威的讲解应该还是一号车马坑入口处博物馆工作人员的讲解。一号车马坑就是最经典的兵马俑坑，有一种千军万马的气势：

#image("005.jpg", width:65%)

另外两个车马坑相对比较小，但是可以近距离看到玻璃柜里面展出的兵马俑，上面丰富的细节清晰可辨。

#image("006.jpg", width:65%)

另外兵马俑景区里面还有一个附属的博物馆，里面展出有一些附近出土的器物。

#image("007.jpg", width:65%)

有一块展板上还有兵马俑的色彩复原图，感觉还挺好看的。可惜这些染料的化学性质并不稳定，一出土就氧化了，没能流传下来。

#image("008.jpg", width:65%)

周边其实还有一个骊山园景区，那个才是秦始皇陵寝真正的所在地，不过因为从兵马俑景区出来有点晚了，所以就没有前往，而是搭乘直达大巴返回西安市区。

= 第3天

今天主要是游玩市内第一天错过的景点。因为今天是工作日，所以小雁塔和西安博物院的票没有那么紧张了，可以很容易预约到。但是陕西历史博物馆还是预约不到，这次只好作罢。

小雁塔原本是一座寺庙，不过看起来已经完全没有佛教活动的痕迹了，仅仅是历史古迹而已。虽然名字是“小雁塔”，不过塔实际上一点也不小。

#image("009.jpg", width: 65%)

除了塔以外，景区里面的一些古代的小雕塑也很有意思。

#image("010.jpg", width: 65%)

小雁塔旁边就是西安博物院，主要是一些古代器物展出。

#image("011.jpg", width: 65%)

从西安博物院出来后的下一站是大雁塔。大雁塔离小雁塔也很近，打个车只花了五块钱，我都有点不好意思，大概是平台有补贴，希望司机拿到手的钱不止这么多。大雁塔在慈恩寺内部，而慈恩寺就是当年玄奘取经回来之后译经的地方。

#image("012.jpg", width: 65%)

进入慈恩寺之后我发现寺庙里面到处都是音箱，不停重复“南无阿弥陀佛”，完全是净土宗的做法，就感觉有点不太对劲。后来查资料才发现，玄奘从天竺回来之后创立的“法相宗/慈恩宗”，自唐武宗会昌毁佛之后，传承已经断绝，经书也大多散失。虽说现在的慈恩寺主打一个玄奘元素，寺庙后面还有个不怎么好看的仿唐建筑“玄奘三藏法师院”；但是基于前述原因，这里其实更像是在慈恩寺原址上建立的新寺庙，从佛教传承的角度，跟玄奘已经没有什么关系了。

#image("013.jpg", width: 65%)

而慈恩寺门口的一条马路就是“大唐不夜城”。白天的大唐不夜城就是一个普普通通的商业步行街，两边的建筑倒是挺有特色，就是质感有点塑料。在KFC随便吃了点东西之后天色渐晚，跟硅基猫看了街边的一个飞行球幕电影，特效还挺好看的，跟迪士尼的相比也毫不逊色。从飞行球幕电影那里出来已经彻底天黑了，旁边的仿古建筑的灯光全部亮起，看起来非常华丽。看起来他们是只考虑了这些建筑在晚上的视觉效果，没有考虑过白天的。看小红书上说晚上这里会有各种各样的节目表演，但是可能因为是淡季，街边走了几趟也没有寻见，只有几个网红的直播舞台看起来很热闹。所以我们就从步行街的另一个出口那里打车回去了。

= 第4天

硅基猫这一天有别的安排暂时离开了。所以今天是我一个人乱逛。我选了一个比较萧瑟的景点：大明宫遗址。

#image("014.jpg", width: 65%)

这个遗址并没有什么特别好看的地方。大明宫的核心区是大明宫的三大殿：含元殿、宣政殿、紫宸殿。历史上这里的三大殿远比北京的故宫豪华，但是时过境迁，大明宫早在黄巢朱温兵乱的时候就已经毁于战火，如今这里更是只剩下了几个夯土台。来这里看看，主要是出于怀古。

  华轩绣毂皆销散，甲第朱门无一半。 \

  昔时繁盛皆埋没，举目凄凉无故物。 \
  内库烧为锦绣灰，天街踏尽公卿骨。 \

= 第6天

第五天没有什么安排，主要就是从西安赶往临汾住下，所以略过。今天的一日游才是重头戏。本来今天是打算跟硅基猫一起的，但是因为一些不可抗力，硅基猫提前跑路了，所以又是我一个人出行。

第一站是洪洞县的广胜寺，广胜寺的琉璃塔是一个很有名的景点，央视的《西游记》电视剧也在这里取景。相比西安单调朴实的大雁塔和小雁塔，广胜寺的琉璃塔异常华丽。只可惜我在西安浪费了太多彩色胶卷，到这里只剩下黑白胶卷了。

#image("016.jpg", width: 65%)

广胜寺分为上寺和下寺。琉璃塔位于上寺，而下寺最出名的是元代的水神庙壁画。壁画似乎用了很厉害的颜料，几百年过去了颜色依然无比鲜艳。

#image("017.jpg", width: 65%)

水神庙旁边还有一个元代的戏台。站在这里可以眺望远处的琉璃塔。

#image("018.jpg", width: 65%)

广胜寺的佛堂里也有悬塑，这是一种北方佛寺的特有的雕塑艺术，主要位于空中。但是悬塑里面，最厉害的还是此行的第二站，隰县的千佛庵，也俗称“小西天”。小西天是一个禅宗院，寺庙建筑小巧精致。这里上千座精美华丽的佛像聚集在一百平米左右的空间里面，震撼程度无以言表。虽然《黑神话：悟空》有诸多为人所诟病的地方，但是能把这么优秀的古代艺术从默默无闻的山西县城推广到大众视野，也是大功一件。因为佛堂内光线昏暗，难以用胶卷拍摄，只能用手机随手拍几张作为旅游纪念。

#image("019.jpg", width: 65%)

最后一站是黄河的壶口瀑布。虽然已经是冬天，但是似乎并没有枯水的迹象，天气也没有冷到河水结冰的程度，因此壶口瀑布得以维持原貌，可以感受到“风在吼，马在叫”的雄壮。黄河的两岸都有壶口瀑布观景台，对面是陕西延安，不过观景的角度感觉不如山西这边。

#image("020.jpg", width: 65%)

= 返程

在临汾休息一晚上后，第二天我就乘坐高铁返回了，一路继续向北抵达太原，然后东出太行山，经过石家庄，到达山东德州，再转车一路向南返回杭州。在路途上还偶遇了这个冬天华北的第一场雪，雪中的太行山格外漂亮。

一周的旅程就此结束。
