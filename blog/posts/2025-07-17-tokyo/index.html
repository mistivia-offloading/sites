在这里开一个串记录一下在东京旅游的经历。

= 第0天

我买的是7月4日到东京的机票，至于为什么是这一天，是因为这一天的机票是最便宜的。后来落地东京了才知道这一天这么便宜是因为有谣传说7月5日日本会发生大地震。现在我们知道这个事件并没有发生，总之赚到。

和国内旅游相比，要带的东西多了这么几样：

- 护照
- 签证
- 流量卡
- 外币现金

国内的手机因为都不支持eSIM，所以比较麻烦，必须要实体卡。虽然据说在机场就可以买到需要的卡，但是为了保险，我还是提前在淘宝上买了SoftBank的流量卡。不能打电话，只能上网，但是已经够用了。

到了成田机场是晚上七点多。本来预期能马上坐上地铁，但是事实上还有一些入关的手续。海关检疫的时候有很可爱的比格犬嗅闻我的行李，真的非常可爱。我很想拍一张照片，但是旁边有牌子写着“禁止拍照”，只好作罢。

按照之前的规划，我需要在机场购买一张Suica交通卡。但是不巧我出来的时候JR的店铺已经关门了，我只能用现金买票。

从成田机场到市区要做的是“京成线”，前往“日暮里”站然后换乘其他线路。这里本来应该乘坐的是“天空快线（SkyLiner）”，但是我对东京的铁路线复杂度没有预期。和国内的地铁轻轨不同，东京的市内通勤铁路往往分为“特急”、“急行”、“各停”，类似于国内铁路上绿皮车的“Z直达”、“K快车”，以及每个小站都会停的慢车。一开始我坐上了慢车，坐着坐着发现不对，就找了一个小站台下车了，开始仔细研究东京铁路的运行逻辑。当时刚好是周五晚上，旁边全部是周五晚上出去狂欢的上班族和学生，非常聒噪，而我一身冷汗在研究这条线路到底是怎么回事，到底有没有做错车。这也是我最紧张的时候。

不过所幸我下车是对的，而且我下车的站点刚好会停靠一辆“急行”的列车，于是我成功抵达了日暮里。

日暮里可以换成到JR线，JR公司也是东京最常用的交通卡Suica的运营公司，我在站点的自动售票机器上买到了一张Suica交通卡。然后就很顺利了。

从铁道站出来已经是晚上十一点多了，东京的治安还是很安全的：下地铁不远就是一所小学，附近有很多监控摄像头，一路上我看到了很典型的日式街道：很多一户建、窄窄的巷子，有点像城中村，但是又很有章法。

这里非常感谢P哥的接待，为了在东京提供了免费的住宿。

因为赶路很累所以早早就睡了，旅程的开始差不多就是这样。

= 第1天

今天是P哥带我游东京。主要活动范围是上野公园。中午在P哥上班地点附近吃了猪排饭，然后路过宝可梦自动贩卖机，购买了一个新叶喵的挂饰。因为天气比较炎热。所以下午第一站是室内的东京国立博物馆。

虽然东京国立博物馆似乎是日本最大的国立博物馆，但是里面的主展览馆的藏品似乎并不怎么样，除了进门处的两尊金刚像比较有气势以外，让我留下印象的东西并不多。但是右侧的“东洋馆”感觉有不少有特色的藏品，很多中国的文物都收藏在这里，甚至还有埃及和两河流域的上古文物。不过这些文物的来历嘛，基本都是黑的，日本人很鸡贼地在文物来源里面写上“某某探险队”，丝毫不提及盗窃乃至抢劫文物的事实。山西的云冈石窟中佛像失去的头部就收藏在这里，不知道还有没有归于原位的可能。

很遗憾，我的新叶喵挂饰的链子不知道什么时候断掉了，遗落在了东京国立博物馆，只留下了一张照片，非常遗憾。

#image("./001.jpg", width: 50%)

从博物馆出来在上野公园附近随便走了走就到了不忍池，一个有很多荷花的小池塘。这种小池塘几乎没有什么特色，昆明市中心的小池塘感觉跟这里的一模一样。

#image("./002.jpg", width: 50%)

东京大学校园就在附近，所以也去看了一下，在著名的安田讲堂大门前拍了几张照片。东京大学校园的建筑非常有特色，都是维多利亚时期的风格，用来拍奇幻题材的电影也不逊色。

#image("./003.jpg", width: 50%)

晚上在东京大学附近吃了一顿咖喱之后，去逛了一下秋叶原。虽然是“二次元圣地”，不过鉴于我现在已经不怎么二次元了，所以没什么感觉。逛了几个动漫题材的商场，感觉跟西湖旁边的二次元商场非常非常相似，我感觉西湖旁边的二次元商业生态应该就是参照秋叶原建立起来的。日本的右翼政治最近似乎在抬头，有右翼政客在街头演讲，欢呼鼓掌的人还挺多的。最后我就在秋叶原买了两本漫画，一本是《剑风传奇》的第一卷，一本是《迷宫饭》的第一卷，用作收藏。

秋叶原有很多赌博店，我去一家老虎机和pachiko店看了一下，围观日本年轻人赌博，感觉他们就在那里不停地机械的按动按钮，就跟斯金纳箱中的小白鼠一样，精神状态非常堪忧。有的老虎机还用了《魔法少女校园》题材作为背景，我作为小圆粉非常不爽。

这一天就是这样。

= 第2天

从这一天开始就是我一个人的东京探索了。

这一天的第一站是涩谷。涩谷其实也就是一个大型的商业街，如果是逛街爱好者的话大概会很开心，不过我对逛街购物实际上兴趣并不是很大，所以也就是随便逛逛。

从涩谷站出来就是著名的“忠犬八公像”。很多人在这里排队和狗雕像合影。还有很多人在这里拍对角线人行道。

#image("./004.jpg", width: 50%)

#image("./005.jpg", width: 50%)

这里的商场经常是一个品牌就是一整栋楼，宜家和MUJI都是这样。不过感觉都没什么好说的。不过逛宜家的时候我顺便在宜家的餐厅里面解决了午饭，然后在MUJI那里买了一个hello kitty的冰箱贴。

下午就跑到新宿御苑去了，感觉这种公园有点像颐和园/玉渊潭。不过即使是周日，人也不是很多，所以可以很休闲地躺在大草坪上望着天空发呆。新海诚的电影《言叶之庭》也在这里取景了，不过其实我并没有看过这部电影，所以也不是很懂。不过风景确实很不错。另外，《你的名字》这部电影里面很有名的一个台阶也在附近，不过逛新宿御苑太累了就没去。

#image("./006.jpg", width: 50%)

晚上回去的时候又路过了一下涩谷，想着去那里的skytree上看看日落。不过门票早就售空了因此没能成行。然后顺便去了一下那里的Pokemon Center，想着再买一个新叶喵的挂饰，结果并没有买到，新叶喵挂饰因为太过火爆已经售空了，非常遗憾。

因为又累又饿，所以没什么心情吃没事，就在麦当劳将就了一下，填饱了肚子就回去了。

第二天就是这样。

= 第3天

这一天的日程是去镰仓。镰仓在东京的南边，比横滨还要远，不过，因为东京便利的通勤系统，加上住的地方本身也靠东京的南部边界，所以只要一个半小时左右就可以到达。

到了JR的镰仓站，可以买到观光电车“江之电”的车票，是800日元，这一天内都可以随便坐车。电车从镰仓出发，一开始的景色平平无奇，就是普通的日本街道，但是突然，电车开到了海边，于是豁然开朗。我下车的第一站是“七里滨”，这里有个很不错的海滩[附件1]，据说是很多动画的取景地。据说天气好的时候这里可以看见海上的富士山，但是那一天空气湿度比较高，能见度并不好，所以实际上并不能看到。

#image("./007.jpg", width: 50%)

第二站是“镰仓高校前”。这里有个面向大海的平交道口，因为是《灌篮高手》中出现的地方，所以很多人在这里拍照。大海、铁道，加上电车，确实有种独特的韵味，难怪有几年国内到处都是号称“小镰仓”的网红景点。

#image("./008.jpg", width: 50%)

第三站是“江之岛”，岛上有座小山，山上有很多神社之类的古迹。但是因为我不怎么熟悉日本历史，所以并不太能能对这个景点提起兴趣。上山的山路两侧有很多小吃，以海鲜为主。

下山之后差不多已经下午三四点，于是我决定返程，回镰仓逛逛。镰仓的街道有点像国内的商业古镇，有很多面向游客的店铺，非常热闹。镰仓的中心是一个神社：鹤冈八幡宫，位于雪之下。据说这个神社本来在电车两站之外的由比滨，后来被源赖朝搬到了这里。日本是用公历过七夕的，所以神社里在举办“七夕祭”，可以看到穿着红白色衣服的巫女在做准备。

#image("./009.jpg", width: 50%)

逛到这里已经快要天黑了，于是折返，回到了东京。

= 第5天

第四天被我跳过了，本来准备在第四天的时候去河口湖看富士山的，但是看了天气预报和直播摄像头，发现富士山最近几天都一直被云雾笼罩，所以非常无精打采，就在房间里蹲了一天，也作为休息。

第五天的第一站是浅草寺，寺庙的建筑很有日式特色。我在浅草寺抽了一个签，很遗憾是凶签，大意是人情处理不好、花开很晚、有鬼来偷钱之类的。然后我就把签系在了旁边的架子上，悻悻而归。

#image("./010.jpg", width: 50%)

第二站是银座，有点像王府井、南京路之类的地方，有很多看起来很奢侈的店，我都不敢去看。银座的十字路口有最近很火的动画《末日后酒店》的那栋大楼。皇居也在银座的附近，但是因为没有预约所以没能进去。这里应该是东京最核心的地段了，但是草丛里却常有流浪汉和乞丐在睡午觉，有点让人意外。

#image("./011.jpg", width: 50%)

第三站是东京塔，远看东京塔的话其实没什么感觉，只是感觉红色的还挺好看的。不过在塔的脚底向上仰望的视觉效果还不错。买一张票可以到塔顶，塔顶可以俯瞰东京湾，东京、川崎、横槟、横须贺都能够在塔顶眺望到，风景非常不错。

#image("./012.jpg", width: 50%)

从东京塔下来差不多天黑了，第五天就是这样。

= 第6天

这一天的目的地是横须贺，也是二战时日军投降后美军登陆的地方。现在则是作为美国海军和日本海上自卫队的军港。美国第七舰队的司令部也在这里。

乘坐JR的横须贺线到达横须贺站之后，出来就能看见一个海滨公园，这里同时也是眺望军港的绝佳地点。我去的时候很不巧，港口内没有航母，但是有美军的“朱姆沃尔特”级驱逐舰，外形非常的科幻，是为了电磁隐身而做的设计。不过根据一些军迷的说法，这艘战舰的性能实际上并不怎么样。港口的另一边停着日本海上自卫队的“小船”，这些就是外观比较传统的战舰了，有高高的桅杆。

#image("./013.jpg", width: 50%)

顺着公园一直走可以到达一个商场。商场里面可以买到“军舰巡游”的票，然后在商场后门的码头可以乘坐游船近距离参观港口内的军舰。绕港口一圈大概是45分钟，船上有一个讲解员，会讲解港口内的每一艘船，只可惜我的日语水平太差，只能听懂偶尔冒出来的几个片假名英语单词，完全不知道他在说什么。

从商场出来沿着公路向东走可以看一看横须贺的街景，这里的到处都是挂着美国国旗的商店，招牌也几乎都是英语，大概是为了服务在这里基地服役的美国军人。走到尽头可以抵达“三笠公园”，这里停靠着日俄战争时期日本联合舰队的旗舰“三笠号”。到了这里能感受到很明显的右翼色彩：公园里面有当时的日本海军大将东乡平八郎的雕像，还有“皇国兴废、在此一战”的标语。作为历史的见证，船我觉得是值得上去一看的，不过里面有很多日本的右翼爱国主义教育的设施，日俄战争也被他们包装成了“亚洲诸民族解放”的战争，让人感到非常不适。想来《进击的巨人》中的“三笠”这个角色就来自这艘战舰吧。

#image("./014.jpg", width: 50%)

这是东京游记的最后一篇了。周五我就去池袋随便看了看，没什么有意思的东西所以不值得写，所以就到这里吧。
